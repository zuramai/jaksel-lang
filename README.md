# Jaksel Lang 

An interpreter inspired by [jaksel-language](https://github.com/RioChndr/jaksel-language)


## Statements

### Assign a variable

```
literally foo itu "hello world" wkwk
```

### Defining a function

```
so about my_story() {
  literally umur itu 22 wkwk
  spill("Umur lu", umur) wkwk
}
```

### Conditions

```
kalo foo itu "hello world" {
  spill("it's hello world") wkwk
}
```

## Expressions

### Mathematical operations

```
1 tambah 2 // 1 + 2
5 kurang 3 // 5 - 3
10 kali 10 // 10 * 10
1 sama dengan 1 // 1 == 1
2 lebih kecil 3 // 2 < 3>
3 lebih kecil sama dengan 3 // 3 <= 3
4 lebih gede 2 // 4 > 2
4 lebih gede sama dengan 4 // 4 >= 4
bukan 1 // !1
```
